Todas las herramientas (CLI) se encuentran ubicadas bajo la carpeta ~/esp/esp-idf/tools

### IDF.PY

```
idf.py
	create-project
	create-component -C components <my-component>
	build
	flash
	menuconfig
	monitor
	set-target esp32
	fullclean
```
